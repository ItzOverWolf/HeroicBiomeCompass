# Heroic Biome Compass

A Minecraft plugin that adds a compass capable of tracking biomes.

## Features

- Locate specific biomes easily.
- Configurable settings for compass behavior.
- Lightweight and performance-optimized.

## Installation

1. Download the latest `.jar` file from [Modrinth](https://modrinth.com/plugin/heroic-biome-compass).
2. Place the file into your server's `plugins` folder.
3. Restart your server.

## Usage

Use the command `/biomecompass` to open the selection menu or track a specific biome.

## Requirements

- Paper/Spigot 1.16+
- Java 17 or higher
